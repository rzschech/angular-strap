Usage

Add a bs-progressbar element to create a progress bar

HTML within the bs-progressbar element will be transcluded.

Options

Options can be passed via data attributes, append the option name to data-, as in data-route-attr="".

Name type default description
value number 0 Current value of the progress bar. Should be between 0 and 100.
animate boolean expression true Toggles whether the progress bar is animated.
type string null Determines the type of the progress bar. Can be success, info, warning, danger, or null for the default type.

Default options

You can override global defaults for the plugin with $progressbar.defaults

        
          angular.module('myApp')
          .config(function($progressbar) {
            angular.extend($progressbar.defaults, {
              animate: false,
              barType: 'warning'
            });
          })
